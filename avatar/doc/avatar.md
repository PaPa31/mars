## Repository inspection

Avatar begins every wake-up by reading the repository's
primary documentation.

When documentary evidence is insufficient, Avatar may use
Git Core to inspect the repository.

Git Core provides low-level repository inspection
capabilities built on Git plumbing.

Git Core is a means of inspection, not a source of
authority.

Repository artifacts remain the primary source of truth.
